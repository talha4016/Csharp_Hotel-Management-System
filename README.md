Hotel Management System is a comprehensive desktop application built with C#, .NET, and a relational database (SQL Server/MySQL). The application automates daily hotel operations by allowing administrators to manage room availability, customer information, reservations, check-in/check-out processes, billing, and staff records. The system ensures secure data management, minimizes human errors, and provides an intuitive user interface for efficient hotel administration.

Key Features

User authentication and role-based login
Room management (Add, Update, Delete, Search)
Guest registration and profile management
Reservation and booking system
Check-in and check-out management
Billing and invoice generation
Employee management
Database integration
Search and filtering
Reports and data management

Technologies Used

C#
.NET Framework / .NET
Windows Forms (WinForms) or WPF
SQL Server / MySQL
ADO.NET / Entity Framework

Hotel Management System

The Hotel Management System is a C# desktop application designed to automate and simplify hotel management tasks. It enables hotel staff to efficiently handle reservations, customer records, room availability, billing, and employee management from a centralized interface.

Features
Secure Login System
Customer Management
Room Management
Reservation Management
Check-In / Check-Out
Billing & Payment
Employee Management
Database Connectivity
Search & Filter Records
User-Friendly Interface
Technologies
C#
.NET Framework
Windows Forms
SQL Server
ADO.NET
Objectives
Reduce manual paperwork
Improve reservation accuracy
Maintain customer and room records
Streamline hotel operations
Generate accurate bills and reports

This project demonstrates object-oriented programming principles, database connectivity, CRUD operations, exception handling, and desktop application development using C# and SQL Server.
